# build
```sh
mkdir build
cd build
cmake ..
make
# or build in docker: 
# - docker run -it --rm -v $(pwd):/project emscripten/emsdk
# - emcmake cmake ..
# - emmake make
npm i
```

# run nodejs
`npm run nodejs`
